# Installation Guide

This guide will help you install and run your React + Vite + Tailwind CSS portfolio project locally on your machine.

## Prerequisites
- **Node.js**: Ensure Node.js (version 18 or above recommended) is installed.
- **npm** or **yarn**: Follow the steps below using your preferred package manager (we'll use npm for this guide).

## Installation

1. **Navigate to the core project directory**
   Open your terminal and make sure you are in the `cy` directory where the application code resides:
   ```bash
   cd c:/Users/chela/OneDrive/Desktop/portfolio/cy
   ```

2. **Install Dependencies**
   Run the following command to download all required packages:
   ```bash
   npm install
   ```

## Running the Application Locally

1. **Start the Development Server**
   Start the Vite dev server by typing:
   ```bash
   npm run dev
   ```

2. **View in the Browser**
   Once the server stats, it will display a local URL (usually `http://localhost:5173/`). Open this link in your web browser of choice.

## Building for Production

If you want to build the optimized version of your app for deployment:

1. **Build the bundle**
   ```bash
   npm run build
   ```
2. **Preview the production build**
   ```bash
   npm run preview
   ```
